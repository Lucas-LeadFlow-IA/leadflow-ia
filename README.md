# LeadFlow IA Pro

SaaS B2B de sales automation propulsé par l'IA Groq.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Framer Motion
- **State**: Zustand
- **IA**: Groq API (Llama)
- **Payments**: Stripe
- **Database**: Supabase

## Getting Started

```bash
# Installation
npm install

# Développement local
npm run dev

# Build production
npm run build
```

## Modules IA

- **BANT** - Qualification de prospects
- **Email** - Rédaction d'emails percutants
- **Script** - Scripts de cold calling
- **Objection** - Gestion des objections
- **CRM** - Fiches contact
- **ROI** - Calculateur de ROI
- **Deal** - Prédiction de deal
- **Sequence** - Séquences email
- **Competitor** - Analyse concurrentielle
- **Proposal** - Propositions commerciales
- **LinkedIn** - Prospection LinkedIn
- **Company** - Veille entreprise
